siground(x,u,sigfigs=3)

Round a measurement x and its uncertainty u to sigfigs. Accepts a single value or a Tuple of values for u.

Returns a Tuple of (x,u...)

e.g. julia> siground(523.12, (15.8,278.0) ) (523, 16, 278)
